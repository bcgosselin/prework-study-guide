# <Prework-Study-Guide-Webpage>

## Description

This project is Robert Gosselin's first web application. This project was created to showcase his skills and aptitude using only text based instructions. This application can be utilized and expanded to act as a reference guide for HTML, CSS, JavaScript, and Git. Everything included was apart of an ongoing bootcamp to develop and hone Full Stack Coding skills.
